# Trabalho de Visualização de Dados - Acidentes em Nova York

Faça o download dos dados de acidentes de carro em Nova York nesse link
https://data.cityofnewyork.us/Public-Safety/Motor-Vehicle-Collisions-Crashes/h9gi-nx95/about_data

Exporte os dados no formato CSV.

Crie uma pasta na raiz chamada data, coloque o arquivo CSV baixado nela.
Renomeie o arquivo para "'Motor_Vehicle_Collisions_-_Crashes.csv"

Execute os comandos

npm install - para instalar os pacotes
npm run dev - para rodar o código

Se tudo estiver correto, será possível ver no browser uma tabela que mostra os 5 distritos e o total de acidentes em cada um.
